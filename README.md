# WhatsApp-bot
My first WhatsApp bot!

*Real-Time News Fetching-
The bot instantly retrieves official news and updates from the website using API calls, ensuring users always get the latest and verified information.

*Automated WhatsApp Responses-
Users can text keywords like “news”, “latest update”, or “info”, and the bot replies instantly with structured updates through Twilio.

*Easy Access to Official Announcements-
All official website updates are automatically delivered in WhatsApp format—no need to visit the site manually.
